Passo para Hospedar um Projeto no GitHub Pages

    1. Crie um repositóro no GitHub com o mesmo nome do seu projeto
    2. No terminal inicialize o repositóro local com git init
    3. Altere o nome da branch princiapl para main
    4. Adicione os arquivos com git add . e faça o commit com git commit -m "firstcommit"
    5. Conecte ao repostiório com: git renote add origin https://github.com/SEU_USUARIO/NOME_DO_REPOSITÓRIO.git
    6. Envie os arquivos com git push -u origin main .
    7. No GitHub, vá em Settings > Pages
    8. Em "Branch", selecione main e a pasta / (root) . clique em Save
    9. Após alguns segundos, seu site estará disponível no link exibido em "Pages"
